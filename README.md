# InitialX

InitialX 是基于 [Initial](https://github.com/jielive/initial) 主题按照自己的喜好二次开发的 Typecho 主题。

演示：[MUKAPP](https://mukapp.top)

只修改了自己用到的部分，如果你发现哪里不契合主题，或者有什么 bug，欢迎 issue。

## 安装

1. 下载 Release 中的 InitialX-vX.X.X.zip
2. 将解压出来的 InitialX 文件夹放到 Typecho 的 `usr/themes/` 目录下
3. 在后台「控制台」→「外观」中启用主题
4. 在「设置外观」中配置主题选项
